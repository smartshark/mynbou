API Documentation
=================

*mynbou* provides a path module which contains :any:`OntdekBaan` which finds every possible path from one node in either direction of the graph via Breadth First Search (BFS) with a configurable breakoff function.
It also contains :any:`Volg` allows following these paths and tracking file renaming and aggregating change information that can then be used to calculate change metrics.
